Apagar todos os dados?

Isto remove permanentemente todo o progresso guardado neste navegador: pontuações, puzzles em curso e estatísticas. Não é possível recuperar depois de confirmar.

💾

Como funciona a gravação

O jogo guarda automaticamente o seu progresso no armazenamento local deste navegador (localStorage), sem enviar nada para a internet. Isso significa que: continua exactamente onde ficou ao reabrir esta página; os dados pertencem só a este navegador e a este computador ou telemóvel; limpar o histórico do navegador ou usar "Limpar dados" apaga tudo, para um recomeço completo.
